# AMBGESTOR 
@author Samuel ALonso

## Descripción
AMBGESTOR es una aplicación para la gestión de dotaciones de ambulancias, optimizando la administración y coordinación de recursos médicos.

## Versión
**1.0.0**

## Tecnologías utilizadas
- **Java**
- **JavaFX**
- **Hibernate ORM**
- **Jakarta Persistence**
- **Maven**
- **MySQL**
